# Icons

Icons come from [Lucide](https://lucide.dev) and are re-exported from
`tajerp_ui/icons` so consumers depend on a single source:

```tsx
import { Search, Plus } from "tajerp_ui/icons";
```

Do **not** install `lucide-react` separately in apps that already use tajerp_ui.

width/height. Match icon size to adjacent text — buttons use `size-4` at `sm`.
