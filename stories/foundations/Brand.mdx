# Brand

The Tijorat+ logo is exported as `<TajERPLogo />` with two variants:

- **`full`** (default) — the green "Tijorat+" wordmark with the "T+" mark. Use in
  app shells, login screens, marketing, and footers.
- **`icon`** — just the rounded-square "T+" mark. Use in collapsed sidebars,
  favicons, avatars, and tight headers.

```tsx
<TajERPLogo />                 // full lockup
<TajERPLogo variant="icon" /> // square mark only
```

Brand colours are baked in (green `#1D9F6E` + white), so the logo reads on both
light and dark surfaces — no `currentColor` theming needed.

## Sizing

The logo is **1em tall** and scales with font-size. Set it via `className`
(`text-[40px]`) or a parent `font-size`; the `full` variant keeps its 396:75
aspect ratio automatically.
