# Typography

**Sans:** Inter (`--tj-font-sans`) — body, UI, tables.
**Display:** Faster One (`--tj-font-display`) — logo and hero headings only.

Import fonts via `tajerp_ui/styles/fonts.css` or the Google Fonts links in your HTML shell.

## Weights

Available as CSS variables: `--tj-fw-regular` (400), `--tj-fw-medium` (500),
`--tj-fw-semi` (600), `--tj-fw-bold` (700), `--tj-fw-black` (800).
