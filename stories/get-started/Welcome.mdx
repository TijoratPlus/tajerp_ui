**tajerp_ui** is the shared React component library and design-token package for
[Tijorat+](https://www.tijorat.plus) products — POS, inventory, analytics, and
back-office tools built on TajERP.

It ships styled primitives (buttons, inputs, tables, overlays), semantic tokens
(`--ui-*` CSS variables), and Tailwind v4 theme registration so every app looks
and behaves consistently.

## What's inside

- **Components** — accessible, brand-styled building blocks built on Radix where needed
- **Tokens** — brand greens, ink scale, status colours, spacing, radii, shadows
- **Icons** — lucide re-exported from `tajerp_ui/icons`
- **Charts** — `AxisChart` wrapper around Recharts (`tajerp_ui/charts`)
